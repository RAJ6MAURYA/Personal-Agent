# Purpose

- The application is for to have a personal local LLM, which remembers you and you can select the right LLM as per need. This also mitigates the idea of leaking the data or using the personal information as the LLM sit's locally in the host machine.
- As the repository grows, more features are expected .

# set up 

1. Install olamma from link: https://ollama.com/
2. Make sure atlease 1 model is downloaded. [ default: llama3.2 ]
3. models can be changed from `modules/backendAI/__init__.py`, so kindly update the model which you like.
4. run `pip3 install requirements.txt`
5. That's it. you're good to go.

# Run

1. execute `python3 main.py`


# Tech Stack

1. Python
2. Ollama
