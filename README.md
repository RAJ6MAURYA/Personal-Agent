# Purpose

- The application is to have a personal localised Agent, which remembers you and you can select the right Model as per your need. This also mitigates the idea of exposing the personal information to internet as the LLM sit's locally in the host machine.
- As the repository grows, more features are expected .

# set up 

1. Install olamma from website - https://ollama.com/
2. Make sure atlease 1 model is downloaded. [ default: llama3.2 ]
3. models can be changed from `modules/backendAI/__init__.py`, so kindly update the file if you want to change the model as per your liking.
4. Run `pip3 install requirements.txt`
5. That's it. you're good to go.

# Run

1. execute `python3 main.py`

# Tech Stack

1. Python
2. Ollama
